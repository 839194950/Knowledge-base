# scaffy

## 概述

scaffy 是一个简单的 Python 脚本，专门用于为任何项目启动 Schema 层！由 ColonelPanicX 构建！它是 [[核心概念/LLM Wiki 基础/LLM Wiki]] 模式的一个很好的起点！

**简单来说：scaffy = 给你的项目快速搭好 LLM Wiki 的架子！**

## 什么是 scaffy？

scaffy 是一个项目脚手架工具，它给你提供一套标准的模板，让 LLM 不需要每次都重新学习规则！

### 核心特点

| 特点 | 说明 |
|------|------|
| **简单易用** | Python 脚本，上手快 |
| **标准模板** | 每个项目有相同的护栏 |
| **Schema 层** | 实现 LLM Wiki 的 schema 层 |
| **好起点** | 适合从零开始 |

### 经典比喻

想象一下：
- 每个项目 = 盖房子
- 传统方式 = 每次重新设计图纸
- **scaffy = 给你一套标准图纸和脚手架！**

## 主要功能详解

scaffy 提供了这些标准模板！

### 1. 协作契约

- 定义协作规则
- 明确角色职责
- 规范沟通方式
- 大家都按规则来！

### 2. 会话协议

- 定义如何对话
- 规范格式
- 统一风格
- LLM 知道怎么配合！

### 3. 看板

- 任务管理
- 进度追踪
- 一目了然
- 项目井井有条！

### 4. 代理配置文件

- 定义 AI 代理
- 配置参数
- 定制行为
- 让代理按你的要求工作！

### 5. 等等

还有更多模板等着你发现！

## 核心理念

scaffy 的核心理念非常重要！

### 每个项目获得相同的护栏

- **护栏** = 标准规则和模板
- **相同** = 每个项目都一样
- **这样 LLM 就不需要每次会话都重新学习规则**

### 为什么这很重要？

让我们对比一下：

**没有 scaffy：**
- 每次新项目 → LLM 重新学习规则
- 效率低
- 容易出错
- 每次都要重新教

**有了 scaffy：**
- 每个项目都有相同的模板
- LLM 已经熟悉这些规则
- 直接开始工作
- **效率大大提高！**

### 这就是 Schema 层！

scaffy 实现了 [[核心概念/LLM Wiki 基础/LLM Wiki]] 中的 Schema 层！这是 Karpathy 所描述的 schema 层的一个很好的起点！

## 与 LLM Wiki 的关系

在 [[核心概念/LLM Wiki 基础/LLM Wiki 三层架构]] 中：
- **The Schema** = scaffy 提供的模板和规则
- **scaffy** = 帮你快速建立 Schema 层的工具

## 适用场景

scaffy 特别适合这些场景：

| 场景 | 为什么适合 |
|------|-----------|
| **新项目** | 从零开始，快速搭架子 |
| **多个项目** | 保持一致性，不用每次重新设计 |
| **团队协作** | 标准模板，大家都遵循 |
| **LLM 开发** | 让 AI 快速上手项目 |

## 如何使用 scaffy？

虽然这是一个简单的 Python 脚本，但使用起来很直观：
1. 运行 scaffy
2. 选择或调整模板
3. 开始你的项目
4. LLM 按标准规则配合！
